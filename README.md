# Dog Benadryl Calculator

A free, SEO-optimized Benadryl dosage calculator for dogs built with Next.js 14.

## Features

- Calculate safe Benadryl dosages based on dog weight
- Support for multiple Benadryl formats (tablets, liquid, chewables)
- Weight unit conversion (lbs/kg)
- Quick reference dosage chart
- Comprehensive safety information and FAQs
- Mobile-responsive design
- SEO-optimized with full meta tags

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Deployment to Vercel

### Option 1: Vercel CLI

```bash
# Install Vercel CLI globally
npm i -g vercel

# Login to Vercel
vercel login

# Deploy
vercel

# For production deployment
vercel --prod
```

### Option 2: Git Integration

1. Push code to GitHub
2. Connect repository to Vercel
3. Vercel will auto-deploy on push
